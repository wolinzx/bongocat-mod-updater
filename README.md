# BongoCat Mod Updater

A Windows desktop utility to download and install BongoCat mod updates automatically.

## Features

- One-click download from Lanzou file hosting
- Automatic RAR extraction and installation
- Simple React-based UI

## Tech Stack

- Electron + React 19 + TypeScript
- Built with electron-vite
- Packaged as a Windows portable `.exe`

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output: `dist/` — portable Windows executable.
